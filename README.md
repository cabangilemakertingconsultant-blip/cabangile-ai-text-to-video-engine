🎬 Cabangile AI Text-to-Video Engine

An intelligent AI-powered video production engine built with Python for Android (Termux), Linux, and desktop environments.

Cabangile AI Text-to-Video Engine automates the creation of narrated videos by generating scenes, images, audio, subtitles, and final video output from a single topic or prompt.

---

✨ Features

- 🎥 AI-powered text-to-video generation
- 🖼️ Automatic scene image generation
- 🎙️ Text-to-speech narration
- 🎞️ Scene-by-scene video rendering
- 📝 Automatic subtitle generation (.srt)
- 📂 File management utilities
- ⚡ Optimized for Android using Termux
- 🔧 Modular provider architecture
- 📊 Logging and progress tracking

🚀 Installation

Clone the repository:

git clone git@github.com:cabangilemakertingconsultant-blip/cabangile-ai-text-to-video-engine.git

Change into the project directory:

cd cabangile-ai-text-to-video-engine

Install the required dependencies:

pip install -r requirements.txt

Run the engine:

python main.py

---

📺 Output

The engine generates:

- Scene images
- Speech audio
- Individual scene videos
- Subtitle (.srt) files
- Final production video

All generated files are saved inside the output/ directory.
